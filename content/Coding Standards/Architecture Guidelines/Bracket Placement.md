## Bracket Placement

- **No extra lines before or after brackets** (`{` or `}`):
  - **Correct:** 
    ```csharp
    public void MethodName()
    {
        // Method implementation
    }
    ```
  - **Incorrect:** 
    ```csharp
    public void MethodName()
    
    {
        // Method implementation
    }
    ```
  - **Incorrect:** 
    ```csharp
    public void MethodName()
    {
    
        // Method implementation
    
    }
    ```
