## Variables Naming Convention
- **Public variables**:
  ```csharp
  public int variableName;
  ```
- **Private variables**:
  ```csharp
  private int _variableName;
  ```
- **Private serialized fields**:
  ```csharp
  [SerializeField] private int variableName;
  ```
- **Local variables**:
  ```csharp
  var variableName
  ```
- **Public properties**:
  ```csharp
  public int VariableName { get; set; }
  ```
